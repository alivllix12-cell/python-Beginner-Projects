# Python Beginner Projects 🐍

A collection of mini Python scripts built to practice core programming fundamentals!

### 🎮 What's Inside:
* `wordcounter.py` - A tool to slice up sentences and count the number of words.
* `madlibs.py` - A funny, dynamic story creator using Python f-strings.
* `numberguess.py` - A numeric logic guessing game against the computer.
* `usernamepass.py` - Practice handling user login variables.
